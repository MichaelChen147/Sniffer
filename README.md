## A net sniffer2.
- 实现一个功能比较简单的、具有图形界面的Sniffer，主线程响应用户界面操作，工作线程完成抓包等工作；
- 够解析出IP层和传输层的协议头，能够过滤TCP、UDP等数据包；
- 能够输出文本方式传送的数据包的内容；
- 能够进行简单的流量统计。


- 开发工具：Intellij IDEA
- 附加库  ：winpcap，jNetPcap
